# Security Policy

## Supported Versions

The following versions are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in Surf Accounting, please do not disclose it publicly. We take security seriously and would appreciate the opportunity to address it before public disclosure.

To report a vulnerability, please email security concerns to the maintainers (e.g., via a private GitHub issue if enabled).

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.
